# 3DFi -- AI Powered 3D Builder Landing Page

A modern and responsive landing page for an **AI-powered 3D asset
creation platform**. This project showcases a clean UI with a hero
section, navigation bar, call-to-action buttons, and a brand/logo
carousel powered by **Swiper.js**.

The page demonstrates a concept platform where users can easily generate
**3D assets using AI** without needing coding skills or powerful
hardware.

------------------------------------------------------------------------

## Features

-   Modern responsive landing page
-   Clean navigation bar with logo and menu
-   Hero section with headline and call-to-action buttons
-   AI product introduction layout
-   Logo carousel using Swiper.js
-   Google Material Icons support
-   Lightweight HTML/CSS structure
-   No framework required

## Swiper Configuration

``` javascript
var swiper = new Swiper(".mySwiper", {
  slidesPerView: 6,
  autoplay: {
    delay: 2500,
    disableOnInteraction: false,
  },
  spaceBetween: 30,
  loop: true,
});
```
